¿Cuál es la diferencia entre autenticación y autorización?
Autenticación: verifica quién es el usuario (ej. login con email y contraseña).
Autorización: verifica qué puede hacer el usuario (ej. acceso a rutas protegidas según permisos).

¿Cuál es la función del token JWT en la guía?
Sirve para autorización, validando que el usuario tenga permiso para acceder a ciertos recursos.
